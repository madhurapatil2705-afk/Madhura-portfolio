Georeferencing historical maps and digitizing spatial features using QGIS to create accurate vector datasets for GIS analysis.

Introduction

This project involved converting scanned maps into spatially accurate digital datasets through georeferencing and digitization. Different geographic features were digitized to create high-quality GIS layers suitable for spatial analysis, mapping, and decision-making.

Study Area

Walwa Subdistrict, Sangli District, Maharashtra, India.

Objectives

  • Georeference scanned maps using Ground Control Points (GCPs).
  • Digitize village boundaries and forest boundaries.
  • Create road, river and land parcel GIS layers.
  • Develop an accurate geospatial database.
  • Prepare high-quality cartographic maps.

Software Used

QGIS
Python
